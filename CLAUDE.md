## 핵심 기능

### 1️⃣ **다중 플랫폼 챗봇**
- **자사몰** (`/chat`) → `/api/chat` 엔드포인트
- **카카오채널** → `/api/kakao` webhook으로 실시간 처리
- **임베드 위젯** (`/widget`) → 네이버, 인포크 등 외부 사이트에 삽입 가능
- 모든 채팅은 `sessionId`로 추적되어 대화 이력 관리

### 2️⃣ **Claude AI 자동응답**
- `lib/claude.ts`의 `getChatResponse()`로 사용자 질문 처리
- 제철삼촌 톤&매너, 상품 지식 프롬프트 기반 응답 생성
- 해결 불가능한 질문에 대해 `escalate: true` 플래그 반환

### 3️⃣ **에스컬레이션 시스템**
- AI가 처리 못하는 문의 → 카카오톡 상담사 자동 알림 (`sendKakaoEscalationAlert`)
- 대시보드에서 "카카오전달" 상태로 표시
- 상담사가 수동으로 처리한 후 "처리완료"로 변경

### 4️⃣ **통합 대시보드** (`/`)
- 실시간 상담 현황 조회 (총/자동처리/카카오전달/처리완료 통계)
- Google Sheets 기반 상담 기록 조회 및 상태 업데이트
- 플랫폼별/상태별 필터링
- 카카오 대기 건 배너 표시 및 카카오톡 일괄 열기

### 5️⃣ **주간박스 시스템** (`/weekly-box`)
- Google Sheets에서 주간 특가 과일 정보 조회
- 재고 상태 표시 (일반/마감임박/매진)
- 카카오톡 자동 알림 발송 가능 (`/api/weekly-box/notify`)

### 6️⃣ **과일 성향 테스트** (`/quiz`)
- 5개 질문 → 8가지 과일 점수 산정
- 최고점 과일 추천 + 상품 링크 제시
- 결과 공유 기능 (모바일 share API 또는 URL 복사)

### 7️⃣ **카카오 OAuth 토큰 관리** (`/kakao-auth`)
- 일회성 카카오 인증으로 `KAKAO_REFRESH_TOKEN` 발급
- 토큰을 `.env.local`에 저장하여 지속적 메시지 발송 가능
