## 🗂 단계별 로드맵

### 1단계 — 미리보기 페이지 + 예약 버튼 (3~4일)
**목표:** 빠르게 띄우기. 핵심 기능만.

- [ ] 주차별 입고 과일 리스트 페이지 제작
- [ ] 과일별 산지 정보 + 사진 표시
- [ ] "지금 예약하기" 버튼 → 기존 주문 페이지 연결
- [ ] Google Sheets 연동 (관리자가 시트에 입력하면 자동 반영)

**담당:** 코더
**리스크:** ✅ 없음

---

### 2단계 — 재고 연동 + 마감 임박 표시 (2일)
**목표:** 희소성 자극으로 전환율 추가 상승.

- [ ] Google Sheets 재고 수량 컬럼 추가
- [ ] 재고 30% 이하 시 "마감 임박 🔥" 뱃지 자동 표시
- [ ] 재고 0 시 "이번 주 마감" 처리

**담당:** 코더
**리스크:** ✅ 없음 (수동 업데이트 방식으로 시작)

---

### 3단계 — 챗봇 + 취향 추천 연동 (2~3일)
**목표:** 기존 프로그램과 시너지.

- [ ] 챗봇에서 "이번 주 뭐 들어와요?" 질문 시 → 미리보기 페이지 연결
- [ ] 취향 추천 결과에서 → 해당 과일이 이번 주 입고 여부 표시
- [ ] "이 과일 이번 주 있어요! 바로 예약 →" 버튼 삽입

**담당:** 코더
**리스크:** ✅ 없음

---

### 4단계 — 카카오 알림 연동 (3~5일, 별도 트랙)
**목표:** 단골 고객 락인.

- [ ] "이 과일 알림 받기" 버튼 → 카카오 채널 구독 연결
- [ ] 입고 전날 자동 알림 발송
- [ ] 마감 임박 시 추가 알림

**담당:** 코더 + 강실장 (카카오 비즈 API 셋업)
**리스크:** ⚠️ 카카오 비즈니스 채널 API 승인 필요 — 별도 진행
**법적 체크:** 수신 동의 없는 대상 발송 금지 (정보통신망법) → 구독 버튼에 동의 문구 필수

```
[하단] 취향 추천 받기 → | 챗봇에게 물어보기 →
```
